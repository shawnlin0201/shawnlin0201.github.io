# function
## function declaration 函式宣告式
```js
function test(){
    //do something
}
```
## function expression 函式表達（運算）式
- Anonymous function expressions 匿名函式表達式。
```js
let test = function () {}
```

- Named function expressions 具名函式表達式。
```js
let test = function something () {}
```
