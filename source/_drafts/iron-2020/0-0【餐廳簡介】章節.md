# 精進程式技術的過程就像是一條不停追逐起點的線

剛踏入這領域的自己，知道自己是多麼的渺小，望著那些看似已成形狀的人，總是不斷地勉勵自己，期望總有一天能和他們一樣。

然而伴隨著無止盡的刺痛與磨難，好不容易撐出了一點模樣，卻發現知道得更多，路途就越遙遠，越難以到達終點。

過了許久，當我停下來檢視過去的痕跡與方向後，這才意識到原來最終我所追逐的目標竟然是原點。
